# Python Projects 🐍

A collection of projects built using Python.  
This repository contains games and utility applications developed for learning, experimentation, and fun.

---

# 📁 Projects Included

## 🎮 Pygame Projects

Projects created using the `pygame` library.

### 1. ClockSystem 🕒
A functional digital clock application built with Pygame.

#### Features
- Real-time clock display
- Clean graphical interface
- Smooth updating system

---

### 2. DinoGame 🦖
A simple dinosaur runner game inspired by the Chrome offline Dino Game.

#### Features
- Jump mechanics
- Obstacle generation
- Score tracking
- Increasing difficulty

---

### 3. FlappyBird 🐦
A clone of the classic Flappy Bird game made using Pygame.

#### Features
- Gravity and movement physics
- Pipe obstacles
- Collision detection
- Score system

---

### 4. PingPong 🏓
A classic two-player Ping Pong game.

#### Features
- Ball physics
- Paddle controls
- Scoreboard
- Multiplayer gameplay
